# 🦠 Sentiment Analysis on COVID-19 Tweets

A Machine Learning and Natural Language Processing (NLP) web application that analyzes the sentiment of COVID-19-related tweets. The model predicts whether a tweet expresses **Positive**, **Negative**, or **Neutral** sentiment through an easy-to-use web interface.

## 🚀 Live Demo

🔗 **Live Website:** https://sentiment-analysis-on-covid-tweets.vercel.app/

## 📂 GitHub Repository

🔗 **Repository:** https://github.com/BUDDEPJASWANTH/sentiment-analysis-on-covid-tweets-project

---

## 📖 Project Overview

This project uses NLP techniques to preprocess COVID-19 tweets and a trained Machine Learning model to classify their sentiment. It demonstrates the practical application of sentiment analysis in understanding public opinions during the COVID-19 pandemic.

---

## ✨ Features

- 📂 Upload a CSV file containing COVID-19 tweets
- 🤖 Predicts tweet sentiment automatically
- 😊 Classifies tweets as Positive, Negative, or Neutral
- ⚡ Fast prediction using a trained ML model
- 💻 Clean and responsive web interface
- 🌐 Deployed on Vercel

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/BUDDEPJASWANTH/sentiment-analysis-on-covid-tweets-project.git
```

### Navigate to the Project

```bash
cd sentiment-analysis-on-covid-tweets-project
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
